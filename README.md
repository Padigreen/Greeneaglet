# Kilishi Delight - Premium Nigerian Spicy Beef Jerky Website

A beautiful, modern, **single-file responsive HTML website** for **Kilishi Delight** by **OJ. Maria Ventures** — showcasing authentic Nigerian spicy beef jerky (kilishi) with a floating WhatsApp order button.

![Kilishi Delight Product Showcase](https://i.etsystatic.com/56142677/r/il/e380aa/6488211117/il_fullxfull.6488211117_j27y.jpg)

## Features

- Fully responsive design — looks great on mobile, tablet, and desktop
- Fixed header with mobile hamburger menu
- Attractive hero section with mouth-watering kilishi background
- Product showcase cards with real packaged kilishi images
- Clear contact information (phone, email, address)
- **Floating WhatsApp button** for instant ordering (pre-filled message)
- Clean, appetizing color scheme inspired by spicy Nigerian flavors
- No external dependencies except Google Fonts
